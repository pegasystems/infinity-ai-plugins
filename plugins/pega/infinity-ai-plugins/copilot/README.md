# Pega Infinity Authoring GitHub Copilot Plugin

This directory contains the source template for the GitHub Copilot plugin variant of Pega Infinity Authoring.

The final packaged plugin is expected to bundle:

- the pinned `infinity-rules-mcp` runtime JAR
- the pinned `infinity-skills` payload
- Copilot-facing setup and authoring skills

This source tree is not the final packaged artifact.
